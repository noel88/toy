## 하루, 하나씩

### **Version 1** (2020.09.01~2020.09.15)
  * 하루에 한번씩 메모할 수 있는 기능
  * 하단 Tab bar로 화면 전환 가능
  * 메모는 에디터로 작성하고 저장 할 수 있다.
  * 메모 확인, 수정, 삭제 가능하다.
  * 메모 전체 삭제를 설정에서 제어 할 수 있다.
  * Playstore Update (2020.09.16)
